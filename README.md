# AI-Powered Stock Price Prediction Platform

## 📌 Overview

AI-Powered Stock Price Prediction Platform is an intelligent financial analytics system that combines Machine Learning, Deep Learning, Sentiment Analysis, and Explainable AI (XAI) to generate stock market insights and BUY/SELL/HOLD recommendations.

The platform collects real-time and historical stock market data, analyzes financial news sentiment, predicts future stock prices using LSTM and Transformer models, and presents results through an interactive Streamlit dashboard.

---

## 🚀 Features

- Real-time and historical stock data collection using Yahoo Finance
- Financial news collection using GNews API
- Technical Indicators:
  - RSI (Relative Strength Index)
  - MACD (Moving Average Convergence Divergence)
  - SMA (Simple Moving Average)
  - EMA (Exponential Moving Average)
  - Bollinger Bands
- Stock Price Prediction using:
  - LSTM Neural Network
  - Transformer Model
- Financial Sentiment Analysis using FinBERT
- BUY / SELL / HOLD Recommendation Engine
- SHAP-based Explainable AI
- Risk Analysis:
  - Daily Returns
  - Annualized Volatility
  - Value at Risk (VaR)
- Interactive Streamlit Dashboard
